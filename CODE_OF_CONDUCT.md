# Code of Conduct

## Expected Behavior

This project should stay practical, respectful, and safe for contributors and users. Please:

- be considerate when discussing bugs, venue data, uploaded photos, and moderation decisions
- keep feedback specific and actionable
- respect privacy and do not post personal information, private messages, secrets, or non-public security details
- avoid harassment, insults, sexualized language, threats, or repeated unwanted contact

## Unacceptable Behavior

Unacceptable behavior includes:

- discriminatory or demeaning comments
- harassment or intimidation
- publishing private information without permission
- knowingly submitting copyrighted official seating charts or other content you do not have rights to share
- posting exploit details publicly instead of using the security reporting process

## Enforcement

Maintainers may edit, hide, lock, or remove comments, issues, pull requests, or uploaded content that violates this code. Repeated or severe violations may result in blocked participation.

For security issues, use the process in [SECURITY.md](SECURITY.md). For other conduct issues, open a GitHub issue with only the necessary context and avoid sharing private information publicly.
